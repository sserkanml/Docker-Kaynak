Docker Nedir?

Docker, uygulamalarınızı hızla dağıtmanızı, paketlemenizi ve çalıştırmanızı sağlayan bir konteynerleştirme platformudur. Konteynerler, uygulama ve bağımlılıklarını bir araya getirerek her ortamda tutarlı bir çalışma sağlar.

Docker Avantajları

  • Hafif ve Hızlı: Docker konteynerleri, sanal makinelerden daha hafif ve hızlıdır, çünkü kendi işletim sistemini paylaşırlar.

  • Taşınabilirlik: Docker konteynerleri, herhangi bir ortamda çalıştırılabilir, bu da uygulamaların her yerde tutarlı bir şekilde çalışmasını sağlar.

  • Yalıtım: Konteynerler, uygulamaları ve bağımlılıklarını izole eder, böylece bir konteynerde yapılan değişiklikler diğer konteynerleri etkilemez.

Nasıl Kullanılır?

  1. Docker'ı resmi web sitesinden indirin ve yükleyin.

  2. Docker'ı yükledikten sonra terminal veya komut istemcisini açın.

  3. Docker konteyneri çalıştırmak için şu komutu kullanın:

   docker run [image_adı]

Docker Komutları

  1. Docker run:

  2. docker run komutu ile docker image'larınızdan container'lar oluşturabilirsiniz.

  # en temel haliyle image çalıştırma:
  docker run nginx
  # Bir konteyneri arka planda çalıştırma:
  docker run -d nginx
  # Bir konteyneri adlandırma:
  docker run --name my_container nginx
  # Port yönlendirme yaparak bir web uygulamasını çalıştırma:
  docker run -p 8080:80 nginx
  # Konteyneri bir ağa bağlama:
  docker run --network=my_network my_image
  # Bir ortam değişkeni ayarlama:
  docker run -e MY_VARIABLE=my_value my_image
  # Konteyneri bir volume'a bağlama:
  docker run -v /host/dizin:/container/dizin my_image

  1. Docker exec:

  2. docker exec komutu, Docker konteyneri içinde çalışan bir komutu yürütmek için kullanılır.

  # Bir komut çalıştırma:
  docker exec my_container ls -l
  # Bir interaktif kabuk başlatma:
  docker exec -it my_container sh
  # Arka planda çalışan bir süreci yönetme:
  docker exec -d my_container nginx -g 'daemon off;'
  # Ortam değişkeni ekleyerek bir komut çalıştırma:
  docker exec -e MY_VARIABLE=my_value my_container echo $MY_VARIABLE
  # Konteynerin başka bir kullanıcı kimliğiyle çalışmasını sağlama:
  docker exec -u www-data my_container whoami

  1. Docker ps:

  2. docker ps komutu, çalışan Docker konteynerlerini listeler.

  # Tüm çalışan konteynerleri listele:
  docker ps
  # Tüm konteynerleri listele (çalışan ve durmuş):
  docker ps -a
  # Özel bir formatta listeleme:
  docker ps --format "{{.ID}}: {{.Image}} - {{.Names}}"

  1. Docker build:

  2. docker build komutu, Docker imajlarını oluşturmak için kullanılır.

     # komutu çalıştırarak Docker imajını oluşturabilirsiniz:
     docker build -t my_image

  1. Docker pull:

  2. docker pull komutu, Docker Hub veya başka bir Docker imaj deposundan bir Docker imajını indirmek için kullanılır.

     # Ubuntu imajını indirmek için:
     docker pull ubuntu
     # Eğer belirli bir sürümü (örneğin 20.04) indirmek isterseniz:
     docker pull ubuntu:20.04

  1. Docker push:

  2. docker push komutu, yerel makinenizde bulunan bir Docker imajını bir Docker imaj deposuna (örneğin Docker Hub) yüklemenizi sağlar.

     # İlk olarak, docker login komutu ile Docker imaj deposuna oturum açmanız gerekir:
     docker login
     # Ardından, docker push komutunu kullanarak imajınızı yükleyebilirsiniz:
     docker push <repository_name>/<image_name>:<tag>

  1. Docker images:

  2. docker images komutu, yerel Docker imaj deposundaki tüm Docker imajlarını listeler. Bu komut, yerel makinenizde bulunan Docker imajlarının bir listesini sunar.

     docker images

  1. Docker login:

  2. docker login komutu, Docker imaj deposuna (örneğin Docker Hub) oturum açmanızı sağlar.

     docker login

  1. Docker logout:

  2. docker logout komutu, Docker imaj deposundan (örneğin Docker Hub) oturumu kapatmanızı sağlar.

     docker logout

  1. Docker search:

  2. docker search komutu, Docker Hub veya başka bir Docker imaj deposundaki imajları aramanızı sağlar.

      docker search <keyword>
      # Örneğin, Ubuntu ile ilgili imajları aramak için:
      docker search ubuntu

  1. Docker network:

  2. docker network, Docker ortamında ağ oluşturma ve yönetme işlemlerini gerçekleştirmek için kullanılan bir komuttur.

    # Ağı Listeleme:
    docker network ls
    # Ağ Oluşturma:
    docker network create my_network
    # Ağ Detaylarını Görüntüleme:
    docker network inspect my_network
    # Ağa Konteyner Ekleme:
    docker network connect my_network my_container
    # Ağı Konteynerden Kaldırma:
    docker network disconnect my_network my_container

  1. Docker volume:

  2. docker volume, Docker ortamında veri saklama ve paylaşma işlemlerini gerçekleştirmek için kullanılan bir komuttur.

      # Volume Listeleme:
      docker volume ls
      # Volume Oluşturma:
      docker volume create my_volume
      # Volume Detaylarını Görüntüleme:
      docker volume inspect my_volume
      # Volume'i Bir Konteynere Bağlama:
      docker run -d -v my_volume:/data my_image
      # Birden Fazla Volume Tanımlama:
      docker run -d -v my_volume1:/data1 -v my_volume2:/data2 my_image

  1. Docker kill:

  2. docker kill komutu, çalışan bir Docker konteynerini zorla sonlandırmak için kullanılır.

      docker kill my_container

  1. Docker logs:

  2. docker logs komutu, belirli bir Docker konteynerinin loglarını görüntülemek için kullanılır.

        docker logs my_container

  1. Docker rm:

  2. docker rm komutu, bir veya birden fazla Docker konteynerini silmek için kullanılır.

         docker rm my_container
         # Birden fazla konteyneri silmek için
         docker rm container1 container2 container3

  1. Docker stop:

  2. docker stop komutu, çalışan bir Docker konteynerini durdurmak için kullanılır.

        docker stop my_container
        # Birden fazla konteyneri durdurmak için:
        docker stop container1 container2 container3

  1. Docker inspect:

  2. docker inspect komutu, belirli bir Docker nesnesinin (konteyner, imaj, ağ vb.) ayrıntılarını JSON formatında görüntülemek için kullanılır.

        docker inspect my_container

  1. Docker history:

  2. docker history komutu, bir Docker imajının oluşturulma geçmişini görüntülemek için kullanılır.

        docker history ubuntu

  1. Docker pause:

  2. docker pause komutu, bir çalışan Docker konteynerini geçici olarak duraklatmak için kullanılır.

        docker pause my_container

  1. Docker unpause:

  2. docker unpause komutu, duraklatılmış bir Docker konteynerini yeniden başlatmak için kullanılır.

        docker unpause my_container
